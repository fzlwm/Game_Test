# 星穹铁道抽卡模块测试用例

## 1. 测试概述
### 1.1 测试目标
- 验证抽卡功能的正确性和稳定性
- 确保抽卡概率符合设计要求
- 测试抽卡结果的展示和奖励发放
- 验证抽卡相关的边界情况和异常处理

### 1.2 测试范围
- 单抽功能
- 十连抽功能
- 保底机制
- 资源管理
- 网络异常处理
- 抽卡概率验证
- UP池机制
- 抽卡记录
- 界面交互

### 1.3 测试环境
| 环境类型 | 具体配置 |
|---------|----------|
| 游戏版本 | v1.0.0 |
| 操作系统 | Windows 10 / iOS 16.0 / Android 13.0 |
| 网络环境 | 稳定网络、弱网络、断网重连 |
| 设备配置 | 最低配置、推荐配置 |

## 2. 测试用例

### 2.1 基础功能测试

#### 2.1.1 单抽功能测试
- **测试ID**：TC-DRW-001
- **测试名称**：单抽一次，验证星琼扣除与物品获得
- **测试类型**：功能测试
- **测试步骤**：
  1. 打开游戏并登录
  2. 进入抽卡界面，选择"限定角色池"
  3. 确保账号拥有160星琼
  4. 点击"单抽一次"按钮
  5. 观察星琼数量变化
  6. 查看获得的物品
  7. 查看抽卡历史记录
- **预期结果**：
  - 成功扣除160星琼
  - 获得一个随机物品（角色/光锥）
  - 抽卡历史中新增一条记录
  - 获得的物品正确发放到背包
