# 黑盒游戏测试作品集

## 项目概述
本项目展示了我在游戏测试领域的专业技能和经验，包含了多个游戏项目的测试案例和缺陷报告。

## 测试理念
- **用户体验至上**：从玩家角度出发，关注游戏的整体体验
- **全面覆盖**：确保测试覆盖游戏的各个功能模块
- **严谨细致**：注重细节，发现潜在的问题
- **持续改进**：通过测试反馈推动游戏质量提升
